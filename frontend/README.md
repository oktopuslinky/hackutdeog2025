# HackUTDE OG 2025 Frontend

This directory contains a React + Vite scaffold ready for building the project UI.

## Getting Started

1. Ensure [Node.js 18+](https://nodejs.org/) is installed.
2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

   The app opens automatically at `http://localhost:5173`.

## Available Scripts

- `npm run dev` — start Vite in development mode with fast HMR.
- `npm run build` — type-check and build a production bundle into `dist/`.
- `npm run preview` — serve the production build locally.
- `npm run lint` — run ESLint with the TypeScript + React ruleset.
- `npm run format` — format the codebase via Prettier.

## Project Structure

- `src/App.tsx` — entry component, includes sample layout and resource links.
- `src/main.tsx` — bootstraps React with StrictMode.
- `src/index.css` — global styles with a simple landing-page aesthetic.
- `vite.config.ts` — Vite configuration with the React plugin.
- `tsconfig*.json` — TypeScript project settings.

Tweak the copy, add components, and integrate APIs as needed.
